Introduction
============

collective.fancyzoom provides smooth javascript image zooming for Plone folders (album view) based on Cabel Maxfield Sasser Fancy Zoom.

See http://www.cabel.name/2008/02/fancyzoom-10.html for further details.

collective.fancyzoom is released under the GPL, but please consider the licence issue of the FancyZoom author.

Installation
============

- install **docbook2sla** either using easy_install or by downloading the sources from the Python Cheeseshop
